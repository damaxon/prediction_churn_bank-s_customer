# Артефакты

В данной директории хранятся артефакты, автоматически создаваемые в процессе обучения модели.

## Содержимое

- Обученные модели (`.joblib`)
- Метаданные модели (`.joblib`)
- Метрики качества
- Результаты подбора гиперпараметров

Файлы в этой папке генерируются автоматически и не предназначены для ручного редактирования.
