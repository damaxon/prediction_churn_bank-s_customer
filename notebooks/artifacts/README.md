# Артефакты ноутбуков

В этой директории хранятся файлы, созданные в процессе работы в Jupyter Notebook (EDA, эксперименты с моделями и анализ результатов).

## Содержимое

- Промежуточные версии моделей
- Экспериментальные метрики
- Вспомогательные JSON-файлы
- Результаты тестовых запусков

Эти артефакты используются исключительно для анализа и экспериментов.

Финальные production-артефакты сохраняются в корневой директории `artifacts/`.
