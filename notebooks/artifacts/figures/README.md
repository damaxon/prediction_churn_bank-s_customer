# Визуализации

В данной папке сохраняются графики и визуализации, полученные в ходе разведочного анализа данных (EDA) и оценки моделей.

## Примеры

- Распределения признаков
- Матрицы корреляций
- ROC-кривые
- Precision-Recall кривые
- Confusion Matrix
- SHAP-визуализации

Все изображения генерируются автоматически внутри ноутбуков и используются для анализа и интерпретации результатов.
