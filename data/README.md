# Данные

В данной директории хранится датасет, используемый для обучения и оценки модели прогнозирования оттока клиентов (churn prediction).

## Источник данных

Датасет взят с платформы **Kaggle** и является публично доступным для использования в учебных и исследовательских целях.

Источник: Kaggle (Bank Customer Churn Dataset)

Данные распространяются в рамках условий использования платформы Kaggle.

## Характер данных

- Датасет не содержит реальных персональных данных.
- Представленные данные являются синтетическими или анонимизированными.
- Информация используется исключительно в образовательных целях.

## Назначение

Данные используются для:

- Разведочного анализа данных (EDA)
- Обучения моделей машинного обучения
- Подбора гиперпараметров
- Оценки качества классификации

## Конфиденциальность

Проект не использует реальные персональные данные клиентов банка.  
Публикация данного репозитория не нарушает требований по защите персональных данных.

Если у вас есть вопросы по лицензированию, пожалуйста, ознакомьтесь с условиями использования датасета на платформе Kaggle.
